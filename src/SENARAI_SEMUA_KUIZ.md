### Fokus Soalan Kuiz:

**Kelantan (12):**
- Kata ganti diri (kawe, demo)
- Kata kerja asas (gi, tubik)
- Kata tugas (hok, bakpo, mano, gak, tokleh)
- Kata adjektif (pahik, bereh)

**Terengganu (13):**
- Kata ganti diri (ambo, mung)
- Kata nama haiwan (tikuh)
- Kata kerja (gi, tubik, kupah)
- Kata adjektif dengan pola mA_A_ (mAneh, mAlAh)
- Kata tugas (hok, bakpe, gok, dok)

**Pahang (17):**
- Fokus PERTANIAN (repih, bakuh, loto, perun)
- Fokus MEMASAK (sabak, balan, sia, merobok, mereneh)
- Kata kerja spesifik (senduh, renceh, cering, roncoh, rembat, kincah, hupor)

**Campuran (15):**
- Perbandingan dialek (bakpo vs bakpe, gak vs gok)
- Persamaan dan perbezaan
- Kata khas setiap negeri
- Pola linguistik (mA_A_ pattern)

## ğŸ’¡ TIPS MENGGUNAKAN KUIZ

### Untuk Pelajar:
1. **Mulakan dengan kuiz negeri** - Kuasai satu dialek dulu
2. **Cuba kuiz campuran** - Selepas belajar semua 3 dialek
3. **Ulang kaji penjelasan** - Baca penjelasan setiap soalan
4. **Fokus pada perbezaan** - Kelantan vs Terengganu (bakpo vs bakpe)

### Untuk Pengajar:
1. **Gunakan progresif** - Mudah â†’ Susah
2. **Highlight pola** - Tunjukkan pattern (mA_A_, -ik ending)
3. **Bandingkan dialek** - Gunakan kuiz campuran untuk compare
4. **Fokus konteks** - Pahang = Pertanian/Memasak, K/T = Asas

### Untuk Penyelidik:
1. **Kesukaran soalan** - Boleh tambah difficulty level
2. **Kategori semantic** - Grouping by meaning
3. **Analisis jawapan** - Track common mistakes
4. **A/B Testing** - Test different question formats
